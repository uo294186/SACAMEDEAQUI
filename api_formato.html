Formato archivo

En cada linea del archivo se debe escribir la cantidad a apostar y el nombre del piloto separados por un guión

Ejemplo: 

            100-Piloto 3
            50-Piloto 3
            80-Fernando Alonso

CUIDADO

La cantidad debe ser un numero positivo y la suma debe ser menor o igual a los puntos maximos

Este archivo sería válido para un jugador con 100 puntos: 

            50-Piloto 3
            20-Piloto 3
            30-Fernando Alonso

Este archivo NO sería válido

            50-Piloto 3
            50-Piloto 3
            100-Fernando Alonso

El nombre del piloto debe ser igual al que se muestra en el juego, respetando espacios y mayúsculas

Si el piloto ganador es el Piloto 2:

Este valor contaría como ganador: 

            50-Piloto 2

Este NO contaría como ganador

            50-piloto2
